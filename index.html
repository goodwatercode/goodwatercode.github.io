01
Windows 11 Migration — 200+ Endpoints
OS migration executed in phased waves with zero unplanned downtime.
MigrationIntuneWindows 11
Overview

Planned and executed the migration of 200+ company workstations from Windows 10 to Windows 11 ahead of end-of-support, across multiple departments and office locations.

Problem

Windows 10 end-of-life was approaching, hardware compatibility varied widely across the fleet, and the business could not tolerate downtime during business hours or loss of user data.

My Role

Sole engineer responsible for compatibility auditing, migration planning, imaging, rollout scheduling, and post-migration support.

Tech Used
Windows 11Microsoft IntuneAutopilotActive Directory
Process

Audited hardware against Windows 11 minimum requirements and flagged devices needing replacement. Built a standardized image and Intune deployment profile. Scheduled migration in waves by department to minimize disruption, with rollback checkpoints at each stage.

Solution

Used Intune for policy-driven, semi-automated upgrades where hardware allowed, and manual clean installs with data migration for edge cases. Ran each wave after hours or during low-usage windows, with a same-day support desk for issues.

Result
200+ devices migratedZero unplanned downtimeOn-schedule delivery
Lessons Learned

Phased, department-by-department rollout with clear rollback points reduces business risk far more than a single "big bang" cutover — and pays off even when it takes longer to plan.

02
IT Network Construction
UPS Replacement, Cable Construcation, Switch/Router Installation, and Server Room Facility Check Up.
IT InfrastructurePowerRack Diagram
Overview

Led a full physical overall of the primary server room: old UPS replacement, new switch and router installation, and a ground-up rack reorganization.

Problem

The existing UPS was end-of-life with degraded battery health, cable runs were undocumented and tangled, and rack space was poorly allocated, making maintenance slow and risky.

My Role

Planned the overhaul, coordinated with vendor and Windows installation , and performed the physical rack, power, and cabling work.

Tech Used
UPS SystemsCisco SwitchesEnterprise RoutersRack & Cable Management
Process

Mapped every existing connection before replacing hardware, sourced a replacement UPS sized for projected 3-year load growth, and staged new switches/routers before a scheduled maintenance window.

Solution

Executed the UPS swap with a temporary bypass to avoid downtime, re-terminated and labeled all cabling to a consistent standard, and re-mounted equipment by function (power, core network, patch panels) for faster future maintenance.

Result
Zero downtime cutover100% cable labelingImproved airflow & access
Lessons Learned

Documenting the "as-is" state before any physical change saves hours later — most server room risk comes from undocumented legacy work, not the new install itself.

03
Cisco IP Phone Deployment
Replaced legacy analog phone lines with a Cisco IP telephony system org-wide.
NetworkingCiscoVoIP
Overview

Deployed a company-wide Cisco IP phone system to replace an aging analog line setup, covering handset installation, VLAN configuration, and call routing.

Problem

Analog lines were costly to maintain, offered no unified directory or voicemail-to-email, and couldn't scale as headcount grew.

My Role

Handled physical installation, network configuration (voice VLANs, PoE budgeting), and end-user handset provisioning.

Tech Used
Cisco IP PhonesPoE SwitchingVoice VLANQoS
Process

Surveyed desk locations and cabling paths, confirmed PoE budget on core switches, segmented voice traffic onto a dedicated VLAN with QoS prioritization, then rolled out handsets desk-by-desk with a fallback line kept live during transition.

Solution

Configured auto-provisioning so new handsets pulled their config on first boot, drastically cutting per-device setup time, and validated call quality under load before decommissioning analog lines.

Result
Full analog decommissionUnified directory & voicemailLower recurring telco cost
Lessons Learned

Voice traffic needs its own VLAN and QoS treatment from day one — trying to retrofit prioritization after user complaints is much harder than designing it in up front.

04
Structured Network Cabling Project
Re-cabled office network infrastructure to an organized, labeled, and documented standard.
NetworkingCablingDocumentation
Overview

Ran a full structured cabling project across office floors, replacing ad-hoc cable runs with labeled, patch-panel-terminated infrastructure.

Problem

Years of incremental changes had left cable runs undocumented, unlabeled, and in some cases run through unsafe paths, making troubleshooting slow and risky.

My Role

Surveyed existing cable plant, designed the new patch panel and labeling scheme, and executed re-termination floor by floor.

Tech Used
Cat6 CablingPatch PanelsCable TestersLabeling Standards
Process

Traced and tested every existing run, retired dead cabling, and re-ran new Cat6 where paths were unsafe or undocumented. Built a consistent labeling convention tied to a patch panel map.

Solution

Terminated all runs to a numbered patch panel scheme matching a floor plan document, tested every port for certification, and handed off a full cabling map for future technicians.

Result
100% ports tested & labeledOn-time, no reworkFull documentation handoff
Lessons Learned

A cabling project's real deliverable isn't the cable — it's the map. Documentation is what turns a one-time cleanup into a maintainable system.

SecurityPalo AltoNetworking
Overview

Assisted in deploying a Palo Alto next-generation firewall to replace legacy perimeter hardware, supporting rule migration and cutover.

Problem

My Role

Tech Used
Palo Alto NGFWSecurity ZonesNAT/RoutingTraffic Logs
